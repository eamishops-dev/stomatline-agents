## §1. Стратегія / позиція
- **Playing to Win** — Martin & Lafley → Where to Play / How to Win; каркас стратегії.
- **Good Strategy / Bad Strategy** — Rumelt → діагноз → напрямна політика → узгоджені дії.
- **Crossing the Chasm** — Moore → beachhead, місто за містом (Київ→Львів→Дніпро).
- **The Innovator's Dilemma** — Christensen → старі дистриб'ютори пропускають AI-native модель.

## §2. Lean / TPS / Theory of Constraints
- **Lean Thinking** — Womack & Jones → value stream, потік, усунення втрат, SOP.
- **Toyota Production System** — Ohno → waste, JIT, kaizen (склад, рух, запаси).
- **The Goal** + **It's Not Luck** — Goldratt (TOC) → вузьке місце, throughput; Revenue Engine (Продажі+Закупки+Фінанси) як одна дошка навколо валового прибутку.
- **Necessary but Not Sufficient** — Goldratt → ERP сама по собі не дає переваги: Odoo + протоколи + AI.
- **The Checklist Manifesto** — Gawande → чеклісти проти помилок (склад/логістика/закупки).
- Toyota Kata — Rother; Managing to Learn (A3) — Shook → щотижневе покращення, проблема → протокол.
- **Lean Enterprise Institute (lean.org)** → відкрита база value stream / Lean / SOP.

## §3. Revenue Engine: продажі / ціна / метрики
- **Monetizing Innovation** — Ramanujam (Simon-Kucher) → ціна як стратегія, а не наслідок; валова маржа.
- **The Balanced Scorecard** — Kaplan & Norton → система метрик по блоках.
- **SPIN Selling** — Rackham; **The Challenger Sale** — Dixon & Adamson → навчати клієнта, не приймати замовлення.
- **The Sales Acceleration Formula** — Roberge → data-driven продажі, playbook, lead scoring.
- The Ultimate Sales Machine — Holmes → системна робота з цільовим списком клінік.

## §4. Jobs to Be Done
- **Competing Against Luck** — Christensen → клініка «наймає» нас, щоб операційний процес не зупинявся.
- **Jobs to Be Done: Theory to Practice** — Ulwick → outcome-driven: міряти результати клініки, не задоволеність.
- Demand-Side Sales 101 — Moesta → trigger events: чому клієнт реально купує.

## §5. Operations / Supply chain / Fulfillment
- **Supply Chain Management** — Chopra & Meindl → мережа, запаси, планування.
- Designing & Managing the Supply Chain — Simchi-Levi; Supply Chain Logistics Management — Bowersox → сервіс-рівень без зайвого cash; дистрибуція.
- **Production and Inventory Control** — Plossl → min/max, safety stock, reorder rules в Odoo.
- **Factory Physics** — Hopp & Spearman → черги, варіабельність, пропускна здатність.

## §6. Метрики / unit economics / масштабування
- **Measure What Matters** — Doerr → OKR: 36-місячна ціль → квартальні цілі блоків.
- **Good to Great** — Collins → flywheel повторного замовлення.
- Scaling Up — Harnish; Traction/EOS — Wickman → операційний ритм малої сильної команди.

## §7. AI-native / продукт / learning organization
- **Co-Intelligence** — Mollick → практика AI-аугментації: AI-помічник на кожній ролі.
- **The Lean Startup** — Ries → build-measure-learn для Блоку A (внутрішній R&D).
- Inspired / Transformed — Cagan → сервіс закупівель як продукт; продуктова модель Блоку A.
- The Phoenix Project / The Unicorn Project — Kim → Odoo+AI+операції як єдина система, flow.
- **Platform Revolution** — Parker, Van Alstyne, Choudary → економіка B2B-платформи, lock-in.
- Machine, Platform, Crowd — McAfee & Brynjolfsson; The Coming Wave — Suleyman → цифрові платформи; governance AI-агентів.
- **The E-Myth Revisited** — Gerber → працювати над системою, не в системі; SOP без героїв.
- The Fifth Discipline — Senge; The Knowledge-Creating Company — Nonaka & Takeuchi; Getting Things Done — Allen → systems thinking; переклад знань засновника в explicit; операційна ясність (Asana).

## §8. Founder mindset / ядро / лояльність
- **The Founder's Mentality** — Zook & Allen (Bain) → інсургент-мислення, одержимість фронтлайном.
- **Profit from the Core** + **Repeatability** — Zook (Bain) → фокус на ядрі: тільки матеріали, тільки клініки 2–8 крісел; повторювані моделі.
- **The Ultimate Question 2.0 (NPS)** + **The Loyalty Effect** — Reichheld (Bain) → switching costs; «клініки, які не можуть уявити роботу без нас» = клієнтська гравітація.
